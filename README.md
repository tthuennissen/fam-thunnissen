# Familienorganizer

Eine einfache Webapp zur gemeinsamen Organisation für Familie und Haushalt.

## Funktionen

- Aufgaben verwalten
- Einkaufsliste pflegen
- Termine planen
- Notizen speichern
- Familienkontakte verwalten
- Daten werden lokal im Browser gespeichert

## Dateien

- `index.html` – Hauptseite der Webapp
- `styles.css` – Styling und Layout
- `app.js` – Logik, Speicherung und UI-Interaktion

## Nutzung

1. Öffne `index.html` im Browser.
2. Füge Aufgaben, Einkaufsartikel, Termine, Notizen und Kontakte hinzu.
3. Die Daten werden automatisch im Browser lokal gespeichert.

## Hinweise

- Die App speichert Daten im `localStorage` des Browsers. Die Organisation funktioniert ohne Server.
- Bei Bedarf kann die App später erweitert werden, z. B. mit Synchronisation, Benutzerverwaltung oder mobilen Funktionen.
